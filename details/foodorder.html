Food Ordering App

A user-friendly web app for restaurants and customers to manage food orders online. It supports real-time order tracking, digital menus, and online payments, built with mobile responsiveness in mind.

Key Features

Digital Menus

Browse and order from dynamic online restaurant menus

Real-time Tracking

Track orders from kitchen to delivery in real time

Secure Payments

Pay online with integrated secure payment gateways

Technologies Used

HTML5 CSS3 JavaScript PHP MySQL

Laundry Booking System

A capstone project that allows users to book laundry services online. Features include account login, pickup and delivery scheduling, real-time laundry status tracking, and seamless payment via e-wallet. Designed to be user-friendly and efficient for both customers and laundry service providers.

Key Features

  • Account login & user dashboard
  • Role-based access for branch riders, branch staff, owners, and customers
  • Pickup & delivery request system
  • Real-time status tracking
  • E-wallet payment support

Technologies Used

HTML5 CSS3 JavaScript PHP MySQL
